# ToDo - App using vue.js

## Project setup
```
A sophisticated ToDO application with vue-router and vuex. it has also api request with axios enabled.

```

### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```
